# notebooks

This is the exploration that produced sigwood's detectors - the FFT beacon scorer,
the DNS clustering, drain3 syslog templating, the CloudTrail dig - kept as runnable
scripts and Jupyter notebooks. They're the working-out, left in the open because the method matters
as much as the result; they aren't maintained as part of the tool. `cloudflaws.ipynb`
runs on the public flaws.cloud dataset, so you can run it yourself (download the
dataset first and point the first cell at your copy). Prototyping a method
here is a welcome way to contribute - see [CONTRIBUTING.md](../CONTRIBUTING.md).
